# The Product Is AI

AI-native infrastructure for AI-enabled teams building AI workflows.

This is a static GitHub Pages site built with plain HTML, CSS, and JavaScript. There is no build step and no framework dependency.

## Local Preview

Open `index.html` directly in a browser.

You can also use a simple local server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

Deploy with GitHub Pages. The `CNAME` file is set to `theproductisai.com`.
